## devTinder 🚀

A full-stack, microservices-based platform for developers to connect, network, and build together. 

This repository documents my **Project-Based Learning** journey through the Namaste Node.js series. The codebase is not static; it evolves and improves episode by episode. You will see the project transform from a raw, initial Express setup into a polished, scalable backend, eventually integrated with a complete interactive frontend.

### 🏗️ Architecture & Tech Stack
* **Frontend:** React.js
* **Backend:** Node.js, Express.js (Microservices pattern)
* **Database:** MongoDB (Mongoose)

### 📂 How to Navigate This Repository (Time Travel)
Because this project improves with every episode, I am using **Git Tags** to track the evolution of the code instead of creating duplicated folders. 

The `main` branch always contains the latest, most advanced version of the project. If you want to see exactly how the code looked at the end of a specific episode, you can time-travel using Git!

**To view a specific episode's code:**
```bash
git checkout ep-<number>
# Example: git checkout ep-4 to see the initial Express setup

```

**To return to the latest working state:**

```bash
git checkout main

```

### 🗺️ The Learning Roadmap

* **Season 1:** Core Node.js internals, Event Loop, Async I/O, and the V8 Engine (Completed).
* **Season 2:** Express.js, MongoDB integration, microservices API design, and building the backend (In Progress).
